# Smart Rural Voting & Participation Analysis System

## Problem
Low voter turnout in rural areas needs data-driven solutions.

## Solution
Built a complete analysis system with Python and Machine Learning.

## Dataset
6 villages with population, votes, literacy rate data.

## Key Findings
- **Village E** has lowest turnout (27.3%)
- **Village F** has highest turnout (85.0%)
- Literacy rate strongly impacts voting behavior

## Machine Learning
- **Model:** Random Forest Regressor
- **Accuracy:** R² Score 0.85
- **Prediction:** Turnout based on literacy + population

- Python (Pandas, Matplotlib, Scikit-learn)
